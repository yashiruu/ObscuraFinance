# Obscura Finance Tracker

Personal Finance Tracker built with the .NET ecosystem as a long-term learning project.

## Vision

Build a personal finance application while learning:

* Clean Architecture
* ASP.NET Core
* Blazor
* EF Core
* SQL Server
* Git Workflow
* Enterprise Development Practices
* LLM Integration
* Agentic AI

## Project Goals

This project has two outputs:

### Product Output

A usable personal finance tracker.

### Learning Output

A deeper understanding of enterprise .NET development.

Topics covered throughout the project:

* Clean Architecture
* Dependency Injection
* EF Core
* REST API Design
* Blazor
* Service Layer
* Repository Pattern
* Unit Of Work
* CQRS
* MediatR
* AI Integration
* Agentic AI
