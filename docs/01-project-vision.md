# Project Vision

## Purpose

Obscura Finance Tracker exists for two reasons:

1. Build a useful personal finance application.
2. Learn enterprise software engineering through a real project.

The application is the vehicle.

The learning journey is the destination.

---

# User Background

- The project started from a personal finance system maintained in Notion.
- The original dataset contains hundreds of real transactions and represents actual financial activity rather than demo data.
- This makes the project significantly more valuable than a tutorial project because every feature solves a real problem.

---

# Why This Project Exists

The project was created to improve understanding of:

* ASP.NET Core
* Blazor
* EF Core
* SQL Server
* Enterprise Architecture
* Software Design
* AI Integration

It also serves as:

* Portfolio project
* Learning laboratory
* Architecture playground
* Long-term technical investment

---

# Learning Philosophy

The project follows a Feature Driven Learning approach.

Instead of:

```text
Study theory for months
    ↓
Start coding
```

The project follows:

```text
Build Feature
    ↓
Understand Flow
    ↓
Refactor
    ↓
Build Next Feature
```

This approach prioritizes practical understanding over theoretical memorization.
